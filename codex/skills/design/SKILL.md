When the user asks about user interfaces, visual design, layout, branding, styling, color, typography, accessibility, or any front-end interaction decision:

1. Do not make design decisions yourself.
2. Use the `design` subagent guidance in `.codex/agents/design.md` to handle the design work.
3. Wait for the design agent's output before implementing.
